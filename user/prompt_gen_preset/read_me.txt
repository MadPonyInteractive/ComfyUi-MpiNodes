A preset is a folder like the one containing this file.
All files in this folder must be un-named and only used for editing their lists.
You can copy this folder to your system to start working on a new preset or use the save preset functionality in a "Mpi RandPromptGen" node.
IMPORTANT: Do not change this files as they are used for the default preset and would break the code, always use a copy on your file system.
